# 🎬 Vidtube – React Application

A YouTube-inspired video streaming web application built using React. It allows users to browse, search, and watch videos in a clean and responsive interface.

---

## 🚀 Features

- Video listing  
- Search functionality  
- Video player page  
- Sidebar navigation  
- React Router navigation  
- Responsive UI  

---

## 🛠 Tech Stack

- React  
- JavaScript  
- React Router  
- CSS / CSS Modules  
- YouTube Data API  

---

## ⚙️ Installation

npm install
npm run dev
